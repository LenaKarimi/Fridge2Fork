# Fridge2Fork

Repo-länk
https://github.com/LenaKarimi/Fridge2Fork.git

Guide till kodstart GUI:
1. Starta via klassen Launcher.java i mappen Launcher.
2. Notera att det ser ut som att koden först kraschar och inte går att köra men det gör den. Minimera Intellij för att se om applikationen dykt upp i bakgrunden, troligtvis har den det.

Guide till kodstart API-connection:
1. Starta Main.java i mappen TheMealDBAPI.
2. Notera att utskrift endast syns i terminalen och inte via GUI.
